# Root Insurance

## Installation

Add hese two lines to your application's Gemfile:

```ruby
gem 'root', git: 'git@github.com:RootBank/root-ruby.git'
gem 'root-insurance', git: 'git@github.com:RootBank/root-insurance-ruby.git'
```

And then execute:

    $ bundle
